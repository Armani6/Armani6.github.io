Hexo 博客部署到腾讯云服务器全流程

2019-08-26

服务器, hexo

由于部署到 github 和 coding 上，访问速度较慢，并且图床 七牛云 自 2018年 7月起，测试域名的生命周期只有 30个自然日，这让维护博客造成了很大的麻烦。

部署环境

  • 本地环境
    • Windows10(64bit)
    • 环境：git，Node.js，hexo…
    • 生成本地静态网站
  • 服务器环境
    • 腾讯云主机(CentOS 7.5 64bit)
    • 环境：git，Nginx，创建 git 用户
    • 使用 git 自动化部署发布

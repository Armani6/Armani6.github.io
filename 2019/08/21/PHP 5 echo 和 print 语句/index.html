在 PHP 中有两个基本的输出方式： echo 和 print。

本章节中我们会详细讨论两个语句的用法，并在实例中演示如何使用 echo 和 print。

PHP echo 和 print 语句

echo 和 print 区别:

  • echo - 可以输出一个或多个字符串
  • print - 只允许输出一个字符串，返回值总为 1

提示：echo 输出的速度比 print 快， echo 没有返回值，print有返回值1。

PHP echo 语句

echo 是一个语言结构，使用的时候可以不用加括号，也可以加上括号： echo 或 echo()。
